How to use jupyter on linux server (windows)

2021-08-02

https://www.digitalocean.com/community/tutorials/how-to-install-run-connect-to-jupyter-notebook-on-remote-server

基本跟这篇一样

先用putty或者别的登录进服务器，再打开
jupyter notebook --no-browser
记住localhost后面跟的端口是多少 (defualt 是8888)

然后再打开putty，选择/SSH/Tunnels，自己电脑的端口(source port)随便选一个（9999），映射的端口(destination)选择localhost后面跟的端口(localhost:8888)。（记得点add）

然后再在自己电脑上打开
http://localhost:9999 (or whatever port number you chose)

就成功了！！

其实第一步只是为了看一下，如果打开jupyter notebook，端口是8888，还是别的，可以只连接后面一个session.
